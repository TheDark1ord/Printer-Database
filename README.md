# Начальная конфигурация проекта
## Подключение необходимых библиотек
В проекте используется несколько библиотек, необходимых для его работы, чтобы их подключить, достаточно
в папке проекта написать консольную комманду
```
composer install
```
если на компьютере не установлен composer, то его скачивание и установка описаны [здесь](https://getcomposer.org/download/).
Для работы сервера так же понадобится apache и php.

## Работа с базой данных
Для начала работы с проектом необходимо задать параметры подключения к базе данных в **config.php**
пример конфига находится в **config.example.php**. После чего можно, подключившись к серверу, вызвать
скрипт **migrate.php**, который позволяет создать в базе данных все необходимые таблицы.

## Конфигурация сервера apache
Для работы приложения понадобится сервер apache и mysql. Для удобной конфигурации сервера можно
[скачать xampp](https://www.apachefriends.org/), который включает в себя несколько модулей,
в том числе и apache и mysql. После установки xampp, неообходимо провести настройку сервера,
чтобы он мог принимать http запросы. Подробный алгоритм настройки можно найти по [ссылке](https://medium.com/@ajtech.mubasheer/setup-https-in-xampp-for-localhost-bc3d01393f31). Можно настроить любой порт, однако frontend расчитывает, что
сервер находится по адресу http://localhost:8080/ для удобства стоит поставить именно этот порт.

# Начало работы
##
Коммуникация с сервером происходит посредствам http запросов, например запрос ***http://server_addr/get/types*** позволит получить
все типы запчастей, заданные в базе данных. Все запросы, которые поддерживает сервер, заданы в файле routes.php.\
Некоторые запросы требуют дополнительных параметров, эти параметры можно увидеть непосредственно в методе, который обрабатывает
этот запрос, на этапе валидации запроса, необходимые параметры заданы так:
```
$validation = $validatior->make($data, [
    'PartName' => 'required',
    'Manufacturer' => 'required',
    'Description' => 'nullable',
    ...
]);
```
чтобы сервер исполнил запрос, его тело должно содержать параметры 'PartName' и 'Manufacturer'.
